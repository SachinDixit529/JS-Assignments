Question 2 : Write the difference between var, let and const with code examples.

In Javascript one can define variables using the keywords var, let or const. var a=10; let b=20; const PI=3.14; The scope of a variable defined with the keyword “var” is global. The scope of a variable defined with the keyword “let” or “const” is limited to the “block” defined by curly braces i.e. {} .
